 Node Version

```
18.16.0
```

1. Install Packages

```
npm install
```

2. Start Project

```
npm run dev
```

## **Options ✍️**

1. Check lint

```
npm run lint
```

2. Fix lint

```
npm run lint:fix
```

3. Check prettier

```
npm run prettier
```

4. Fix prettier

```
npm run prettier:fix
```

5. Fix lint and prettier

```
npm run format
```
